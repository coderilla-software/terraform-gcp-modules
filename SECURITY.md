# Security Policy

## Supported Versions

This is an open-source project maintained on a best-effort basis.

## Reporting a Vulnerability

If you believe you have found a security issue, please do **not** open a public issue.

Instead, email: **security@coderilla.de**  
(If you prefer, you can also contact us via GitHub private message.)

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes

We will respond as soon as possible.
